Meet CaptainCore

CaptainCore is my WordPress management toolkit

  • It's open source:
    https://github.com/CaptainCore/
  • It's running in production:
    https://anchor.host/my-account/

The code is broken out in 3 seperate projects.

Some similarities to ManageWP.

Built to solve my needs as a WordPress maintenance provider.

Warning ‚ö†

The following is a work in progress. Code is publically available. Only ALPHA has been released.

Feature: Quicksaves

  • Solves the problem "What changed and when".
  • Easily rollback back previous versions of themes/plugins.

Feature: Site Monitor

  • Built to be automattic and fully integrated.
  • Single email per check. Not 500!

Feature: Downloadable snapshot

  • Platform independent.
    (Works with Kinsta and WP Engine)
  • Efficient incremental sync via SFTP

Feature: Login As

  • Quick login as any user.
  • Works with both production/staging environments.
