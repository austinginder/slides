## 1. Someone Bought 30 WordPress Plugins and Planted a Backdoor in All of Them

- A malicious actor purchased the Essential Plugin portfolio on Flippa for six figures in early 2025
- Within months, injected a sophisticated backdoor into 30+ plugins
- Attack remained dormant for 8 months before activating in April 2026
  - Distributed SEO spam and hidden content to search engines
  - Used blockchain-based command-and-control servers
- 31 Essential Plugin titles permanently closed from WordPress.org
  - Countdown Timer Ultimate, Popup Anything on Click, WP Testimonial with Widget, and others
- Discovered by Austin Ginder of Anchor Hosting
- Highlights a critical gap: WordPress.org has no mechanism to flag or review plugin ownership transfers
- Takeaway: Site owners should audit installations and inspect wp-config.php for injected backdoor code

Source: https://anchor.host/someone-bought-30-wordpress-plugins-and-planted-a-backdoor-in-all-of-them/

## 2. EmDash: Cloudflare's New CMS

- Interview with the 2 Matts from Cloudflare on WP Product Talk (Episode 133)
  - **Matt "TK" Taylor** - Senior Product Manager, previously at Financial Times and News Corp
  - **Matt Kane** - Senior Principal Engineer, core maintainer of the Astro web framework
- EmDash is a new CMS positioned as a "spiritual successor" to WordPress
  - Built with Astro framework and Node.js
  - Described as "what WordPress would be if it was created in 2026"
- Key differentiators from WordPress
  - Plugin sandboxing: plugins have granular, specified capabilities rather than unrestricted access
  - Built-in SEO and role management instead of relying on plugins
  - AI-native design emphasizing developer experience
  - MIT licensed instead of GPL
  - Runs on standard Node.js infrastructure (Cloudflare Workers, AWS Lambda, Netlify, Vercel)
- Opportunity: early "gold rush" for plugin developers building first-mover solutions
- Threat: addresses WordPress pain points like security and modern development practices
- Open-source, community-driven governance structure under consideration

Source: https://wpproducttalk.com/podcast/emdash-new-plugin-opportunity-or-threat/

## 3. WordPress 7.0 Release Delayed

- Originally targeted for approximately April 9, 2026
- Progressed through Beta 1-4 (Feb-March) and entered Release Candidate phase March 19
- Release Candidate 3 and final release dates marked as "Delayed"
- A "Extending the 7.0 cycle" post was published March 31
- Updated schedule expected to be published no later than April 22
- Possibly releasing later this month

Source: https://make.wordpress.org/core/7-0/
