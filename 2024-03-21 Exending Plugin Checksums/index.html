Extending Plugin Checksums

 

by Austin Ginder

What are checksums?

- A checksum is a string of numbers and letters that acts as a fingerprint for a file. - A way to verify files haven't been tampered with.

Verifying checksums with WP CLI

`wp core verify-checksums` `wp plugin verify-checksums --all`

Only WordPress.org plugins are verified

[Example of WooCommerce v8.6.0 checksums](https://downloads.wordpress.org/plugin-checksums/woocommerce/8.6.0.json)
